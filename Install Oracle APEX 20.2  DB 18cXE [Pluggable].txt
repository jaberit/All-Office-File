a. Unzip it where you want to Install Oracle Database [XE] 18c
b. Unzip "apex_20.2" to the folder where you want to install it, so you do not need to copy & paste it (apex folder)
c. Unzip ords 20.3 as well.
d. Run database setup with "Administrator" rights.
e. Hit "Next" button.
f. Check on "I accept the terms..." & hit "Next" button.
g. Change "Directory" & hit "Next" button.
h. Enter Database Password & Confirm Password from your choice, hit "Next" button.
i. Not hit "Install" button.
j. Hit "Finish" button.
Database [XE]18c Installation has completed Pluggable Database name is "XEPDB1"
k. Open command prompt with "Administrator" rights & reach where apex folder is available.
l. Run sqlplus /nolog
m. Connect as sys as sysdba
n. Find pluggable database name run this command
show pdbs;
o. Run this command
ALTER SESSION SET CONTAINER = XEPDB1;
p. Now install apex by running this command
@apexins sysaux sysaux temp /i/
q. Run @apxchpwd.sql, username is "ADMIN", email, & password from your choice.
r. Run @apex_rest_config.sql (provide password for APEX_LISTENER, & APEX_REST_PUBLIC_USER from your choice)
s. Open SQL Plus connect with sys / as sysdba run these commands.
ALTER SESSION SET CONTAINER = XEPDB1;
ALTER USER APEX_PUBLIC_USER IDENTIFIED BY oracle ACCOUNT UNLOCK;
t. Open Command prompt again with "Administrator" rights & reach where ords is available.
u. Run this command java -jar ords.war install advanced.
v. Provide Configuration path & hit "Enter" button. "C:\18cXE_20.2\ords"
w. Enter number for... [1]: Hit "Enter" button.
x. Database server is [localhost]: Hit "Enter" button
y. Enter listen port [1521]: Hit "Enter" button, my port is 1521 your port will be different.
z. Specify Database SID [1]: provide option "1" & hit "Enter" button, my database service name is "XEPDB1"
za Enter 1 if you want to verify... [1]: Hit "Enter" button.
zb. Enter password & Confirm password for ORDS_PUBLIC_USER
zd. Enter administrator username, password, & Confirm password.
ze. Enter the default... [SYSAUX]: Hit "Enter" button.
zf. Enter the temporary... [TEMP]: Hit "Enter" button.
zg. Enter the default... ORDS_PUBLIC_USER [USERS]: Hit "Enter" button.
zh. Enter the temporary... ORDS_PUBLIC_USER [TEMP]: Hit "Enter" button.
zi. If using Oracle Application... [1]: Hit "Enter" button
zj. Enter the PL/SQL Gateway... [APEX_PUBLIC_USER]: Hit "Enter" button.
zk. Specify password, & Confirm password for APEX_PUBLIC_USER from your choice.
zl. Enter 1 to specify passwords... [1]: Hit "Enter" button & specify password, & Confirm Password for both "APEX_LISTENER" & "APEX_REST_PUBLIC_USER" from your choice.
zm. Enter a number to select a feature to enable [1] SQL Developer Web... [1]: Hit "Enter" button.
zn. Go to apex folder copy "images" folder from "C:\18cXE_20.2\apex" & paste it to "C:\18cXE_20.2\ords"
zo. Enter 1 if you wish... [1]: Hit "Enter" button
zp. Enter the APEX static resources...: Provide "image" path "C:\19c_20.2\ords\images"
zq. Enter 1 if using HTTP.... [1]: Hit "Enter" button.
zr. Enter the HTTP Port [8080]: Hit "Enter" button or define from your choice.
zs. Open cmd with administrator rights, reach where ORDS folder is available (C:\19c_20.2\ords) & run this command "java -jar ords.war", 
when "Oracle REST Data Services initialized" appear use Oracle APEX, minimize command prompt not close it.
